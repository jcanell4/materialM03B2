Exercici 1 - Ús d'arrays com a paràmetres d'entrada o sortida

Els arrays són un tipus de dada compost. Com sabeu, un array referencia tota una col·lecció de dades, i disposa de mecanismes per accedir i manipular els seus elements. La gran avantatge dels arrays és que amb una única variable podem accedir a multitud de dades. De fet, són també un mecanisme per reduir la complexitat. Es mantenen el nombre de dades, però s'aconsegueix reduir el nombre de variables cosa que facilita la implementació del programador.

De forma semblant a l'ús d'arrays per mitjà de variables, és possible fer servir paràmetres de funcions que referenciïn també arrays, de manera que sigui possible accedir a tots els seus elements utilitzant només un únic paràmetre

En aquest primer exercici se us proposa que practiqueu a crear funcions que rebin arrays. Concretament us proposem que feu tres funcions que rebin arrays de tipus double.

La primera s'anomenarà valorMitjà i calcularà el valor mitjà de la col·lecció de doubles que li arribi a través d'un paràmetre de tipus array. La funció haurà de retornar el valor calculat

La segona funció s'anomenarà valorMinim i en aquest cas, cercarà el valor mínim de tots els doubles que arribin per paràmetre. Per tant la funció retornarà un double amb el valor mínim trobat

La darrera funció s'anomenarà posicioMaxim. Es tracta d'una funció que tindrà per objectiu trobar el valor màxim dins l'array que li arribi per paràmetres, però en comptes de retornar el valor, el que retornarà és la posició de l'array on es trobi l'element amb el valor màxim. Retornar la posició en comptes del valor pot tenir certes avantatges; sovint cal treballar amb diversos arrays relacionats a través de la posició, per exemple per emmagatzemar les notes i els noms dels alumnes cal fer servir dos arrays perquè les dades son se tipus diferents. En un array de cadenes de text hi guardarem els noms dels alumnes i en un altre de doubles les seves notes. Ambdós arrays els mantindrem relacionats per posició. És a dir, el primer alumne tindrà la primera nota, el segon la segona nota i així successivament. Saber la posició d'un element especific d'un dels arrays suposa també disposar del valor corresponent de l'array realcionat. És per això que quan programen amb arrays pot ser preferible treballar sempre amb les posicions i fer el canvi només quan sigui necessari.

    double valorMitja(double[] valors){
        double vSuma = 0;
        for(int i=0; i<valors.length; i++){
            vSuma+=valors[i];
        }
        return vSuma/valors.length;
    }
    
    double valorMinim(double[] valors){
        double vMinim = valors[0];
        for(int i=1; i<valors.length; i++){
            if(valors[i]<vMinim){
                vMinim=valors[i];
            }
        }
        return vMinim;
    }
    
    int posicioMaxim(double[] valors){
        int pMaxim = 0;
        for(int i=1; i<valors.length; i++){
            if(valors[i]>valors[pMaxim]){
                pMaxim= i;
            }
        }
        return pMaxim;
    }
}
