Exercici 1 - Ús d'arrays com a paràmetres d'entrada o sortida

Els arrays són un tipus de dada compost. Com sabeu, un array referencia tota una col·lecció de dades, i disposa de mecanismes per accedir i manipular els seus elements. El gran avantatge dels arrays és que amb una única variable podem accedir a multitud de dades. De fet, són també un mecanisme per reduir la complexitat. Es mantenen el nombre de dades, però s'aconsegueix reduir el nombre de variables cosa que facilita la implementació del programador.

De forma semblant a l'ús d'arrays a la declaració de variables, és possible fer servir paràmetres de funcions que també referenciïn arrays, de manera que sigui possible accedir a tots els seus elements utilitzant només un únic paràmetre

En aquest primer exercici se us proposa que practiqueu la creació de funcions que rebin arrays. Concretament us proposem que feu tres funcions que rebin arrays de tipus double.

La primera s'anomenarà valorMitjà i calcularà el valor mitjà de la col·lecció de doubles que li arribi a través d'un paràmetre de tipus array. La funció haurà de retornar el valor calculat

La segona funció s'anomenarà valorMinim i en aquest cas, cercarà el valor mínim de tots els doubles que arribin per paràmetre. Per tant la funció retornarà un double amb el valor mínim trobat

La darrera funció s'anomenarà posicioMaxim. Es tracta d'una funció que tindrà per objectiu trobar el valor màxim dins l'array que li arribi per paràmetres, però en comptes de retornar el valor, el que retornarà és la posició de l'array on es trobi l'element amb el valor màxim. Retornar la posició en comptes del valor pot tenir certes avantatges; sovint cal treballar amb diversos arrays relacionats a través de la posició, per exemple per emmagatzemar les notes i els noms dels alumnes. En aquest cas caldrà fer servir dos arrays perquè les dades son de tipus diferent. En un array de cadenes de text hi guardarem els noms dels alumnes i en un altre de doubles les seves notes. Ambdós arrays els mantindrem relacionats per posició. És a dir, el primer alumne tindrà la primera nota, el segon, la segona nota i així successivament. Saber la posició d'un element especific d'un dels arrays suposa també disposar del valor corresponent de l'array relacionat. És per això que quan programen amb arrays pot ser preferible treballar amb les posicions i obtenir el valor només quan sigui necessari.

        double vSuma = 0;
        for(int i=0; i<valors.length; i++){
            vSuma+=valors[i];
        }

    }
    
    double valorMinim(double[] valors){
        double vMinim = valors[0];
        for(int i=1; i<valors.length; i++){
            if(valors[i]<vMinim){
                vMinim=valors[i];
            }
        }
        return vMinim;
    }
    
    int posicioMaxim(double[] valors){
        int pMaxim = 0;
        for(int i=1; i<valors.length; i++){
            if(valors[i]>valors[pMaxim]){
                pMaxim= i;
            }
        }
        return pMaxim;
    }
}

Ara bé, les funcions, a més de retornar arrays arribats a través de paràmetres, també poden instanciar nous arrays, donar-los valors i retornar-los. Així, és habitual fer servir funcions per inicialitzar arrays amb valors per defecte o per crear instàncies d'arrays dels que desconeixem la seva mida en el moment de codificar, però que durant l'execució sigui possible calcular-la.


public class FuncionsIArrays {
    public static void main(String[] args) {
        //adapteu la instanciació del programa d'acord amb el fitxer on treballeu
        FuncionsIArrays prg = new FuncionsIArrays();
        prg.inici();
    }

        /* retorn d'arrays*/
        int[] varArray;

    int[] retornarArrayInicialitzat(){

        }
        return valorARetornar;
    }

        for(int i=0; i<limitFor; i++){
            System.out.print(valors[i]);        
            System.out.print(", ");        
        }
        if(limitFor>=0){
            System.out.print(valors[limitFor]);        
        }
        System.out.println(" }");        
    }

}

Com continuar

Un cop acabada aquesta activitat, heu de ser capaços d'enfrontar-vos a aplicacions molt més complexes que les que heu fet fins ara. L'ús d'arrays a les funcions permet reduir dràsticament la complexitat sempre que es tinguin en compte les precaucions vistes aquí: els arrays inicialitzats passen a les funcions per referències, malgrat que una instanciació d'un paràmetre de tipus array dins una funció esborraria totalment el vincle referencial amb la variable original. Sortosament és possible retornar arrays a les funcions. El retorn d'una array es fa també per referència. Retornar arrays, és una bona manera d'accedir als canvis d'un array instanciat dins la mateixa funció.
